搭建个人博客的历程

Posted on 2020-09-26

date: 2020-09-26 09:21:02
#首先本来想用webstorm来用代码创造一个博客,但是webstorm没法破解并且看了HTML标签后就放弃了(本身是个萌新)

#于是又开始百度,经过n小时的不断寻找后.就打算用自己较为熟悉的frontpage..在看了许多国内著名博客网站例如网易,CSDN后,并没弄懂页面的原理,于是又放弃…

#在不断的百度中,又看到了PS制作法,但是考虑到丝毫没有技术含量,害怕大佬们鄙视,就也放弃了.

#最终在CSDN博客中搜寻时,看见了Github与hexo制作博客(可能有那么一点点技术含量)也就开始钻研了.首先学习了如何将github与git连接起来(过程略),github是承载hexo博客的载体,应该首先学习.其次安装Node来辅助安装插件,最后再安装hexo,终于完成了相关环境的配置.

#然后在git界面输入hexo init chzvblog来创建博客,再输入npm install初始化博客,再输入hexo g ,hexo s来测试博客是否创建成功,然后开始将博客部署到github.首先创建账户和仓库,再创建github-pages,再在hexo文件夹中打开_config.yml文件。

#将其中的

#Deployment

#Docs: https://hexo.io/docs/one-command-deployment

#下方加入

#deploy:

type: git

repo: https://github.com/gapman123/123

branch: master

#设置完后再输入npm install hexo deployer git --save,于是再输入hexo clean,hexo g,hexo d,完成部署,即可输入https: //用户名.github.io/仓库名/打开博客,

#最后个人认为github自带主题不好康,于是换了一个其他大佬的主题,先找到相关主题,再打开它的github主页,在git中输入git-clone-网址,再在hexo中_config.yml,theme后面改为主题名.在theme文件夹中_config.yml,选择主题种类最后将hexo中_config.yml中URL网页地址改为自己github网页地址,root后改为/仓库名/,再将.deploy_git文件夹删去,再在git中输入hexo-clean,hexo-g,hexo-d,完成操作(最后改主题真的耗了好长时间，可能还是不熟练).

#最后希望web组的大佬不要嘲笑小白(我自己)的lj博客,希望能加入web组和各位大佬们一起学习技术攻克难关.
